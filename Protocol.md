# Splix.io protocol

## Servers
Servers are located by json file http://splix.io/json/servers.json

## Client

| Packet name | Data type | Name in game     | Description
|:-----------:|-----------|----------------- | ------------
| 1           |           | UPDATE_DIR       | Update direction
| 2           |           | SET_USERNAME     | Set username
| 3           |           | SKIN             | Set skin
| 4           |           | READY            |
| 5           |           | REQUEST_CLOSE    |
| 6           |           | HONK             |
| 7           |           | PING             | Ping server
| 8           |           | REQUEST_MY_TRAIL |
